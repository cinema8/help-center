# Iframely Integration with Cinema8

## How to Embed Cinema8 Videos Using Iframely API

**Iframely** is a popular digital content embedding API that provides responsive web embeds from over 1,900 verified media publishers. It offers rich media previews, responsive iframes, and metadata for use in editors, CMS platforms, and third-party publishing tools.

<Callout type="info">
Cinema8 is an officially supported media publisher in the Iframely ecosystem. That means your interactive videos can be automatically previewed and embedded by any Iframely-powered platform.
</Callout>

### What You Can Do with Iframely

- Generate responsive embed previews of Cinema8 videos across platforms
- Enable automatic media detection and formatting in CMS tools
- Deliver rich metadata for preview cards and video context

### How to Embed a Cinema8 Video via Iframely

Simply paste your interactive video’s public URL into any Iframely-enabled editor or CMS.

```text
https://cinema8.com/video/YOUR_VIDEO_ID
```

Iframely will handle all rendering automatically, pulling metadata like:

- Title and thumbnail
- Embed dimensions
- Interactivity support via iframe

<Callout type="important">
There’s no need to manually generate embed codes when using platforms that support Iframely. Paste the link, and the preview will appear instantly.
</Callout>

## Where You Can Use It

Cinema8 videos embedded via Iframely work seamlessly in:

- Headless CMS platforms (e.g., Strapi, Contentful with Iframely plugins)
- Blog engines like Ghost or Medium (when Iframely proxy is configured)
- Internal knowledge base tools or social platforms supporting embed previews
