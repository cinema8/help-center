# SCORM-compatible Exports for LMS

<Callout type="info">
Cinema8 lets you create SCORM 1.2 and 2004-compatible interactive video packages for any LMS that supports SCORM standards.
</Callout>

## Use Cinema8 as a Next-Generation SCORM Authoring Tool

Cinema8 supports:

- **SCORM 2004**

Compatibility is manually verified across major LMS platforms by our learning technology experts.

## How to Export SCORM Packages

<Steps>

### Step 1: Open Sharing Options

Navigate to the **Quick-start dashboard** of your interactive video. Click on **Sharing Options**.

### Step 2: Select SCORM Export

In the popup window, go to the **SCORM Export** section.

![SCORM Export Option](/screenshot/scorm/scorm-export-button.webp)

### Step 3: Configure Export Settings

In the export options modal:

- Set a **minimum time for completion** (in seconds).
- Choose your desired **SCORM version**.
- Optionally, select a **subtitle track** to include.

![SCORM Export Settings](/screenshot/scorm/scorm-export-settings.webp)

### Step 4: Download the SCORM Package

Click **Export**, and a `.zip` file will be generated. You can now upload this file to any SCORM-compliant LMS.

</Steps>

<Callout type="important">
Cinema8 exports are ideal for Moodle, Blackboard, Canvas, TalentLMS, and any SCORM-compliant system.
</Callout>
