# How to Use Audio Element

<Callout type="info">
The **Audio Element** allows you to add background music or sound effects to enhance the mood and impact of your interactive videos.
</Callout>

## Adding an Audio Element

![Adding Audio Element](/screenshot/advanced-interactivity/audio-element-1.webp)

After inserting the Audio element into your timeline, it will appear in the layer view.

<Steps>

### Step 1: Browse the Sound Library

Cinema8 provides a built-in sound library with royalty-free music and effects.

### Step 2: Select Your Audio

Choose the desired file from the library.

### Step 3: Download or Preview

You can preview or download the audio to your local machine if needed.

### Step 4: Apply Audio

Click **Save** to add the selected audio to your timeline.

</Steps>

## Audio Settings Panel

Once added, clicking the element in the timeline opens its properties panel.

![Audio Element in Timeline](/screenshot/advanced-interactivity/audio-element-2.webp)

### Basic Settings

| Option       | Description |
|--------------|-------------|
| **Name**     | Assign a label for the element. |
| **Start Time** | Defines when the audio begins playing. |
| **Preview**  | Play back the audio for quick review. |
| **Replace Audio** | Choose a different file from the library. |

![Audio Settings Panel](/screenshot/advanced-interactivity/audio-element-3.webp)

### Playback Controls

| Option       | Description |
|--------------|-------------|
| **Auto Play** | Enable or disable playback at start time. |
| **Force Play** | Plays the audio even if not triggered, useful for multi-language videos. |
| **Pause with Video** | Audio pauses when the video does. |
| **Loop Audio** | Repeats the audio continuously during its active time. |
| **Stop on Question** | Automatically stops the audio when an interactive question appears. |

### Action Triggers

You can assign actions to fire **when the audio element appears or disappears**:

| Trigger              | Description |
|----------------------|-------------|
| **On Appear**        | Runs one or more actions when the element becomes visible. |
| **On Disappear**     | Runs actions after the element ends. |
| **+ Add Action**     | Adds additional triggers per phase. |

<Callout type="default">
For a full list of supported actions and how to configure them, see the [Element Actions Reference](/design-advanced-interactivity/element-properties-common-features).
</Callout>

### Final Options

| Option       | Description |
|--------------|-------------|
| **Save**     | Saves all settings and changes. |
| **Delete**   | Removes the audio element from the timeline. |
