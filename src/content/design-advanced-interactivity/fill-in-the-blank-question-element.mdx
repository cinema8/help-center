# Fill in the Blank Question Element

<Callout type="info">
This element allows viewers to type answers into blank spaces embedded in your video’s content — ideal for comprehension checks or open-ended prompts.
</Callout>

## How to Add and Configure a Fill-in-the-Blank Question

<Steps>

### Step 1: Add the Element to the Timeline

Drag and drop the **Fill in the Blank Question** element from the Elements list to your video layer.

![Drop Element](/screenshot/questions/fillintheBlankQ1.webp)

The configuration panel will appear:

![Initial Panel](/screenshot/questions/fillintheBlankQ2.webp)

1. Enter the question text  
2. Insert a blank field  
3. Add general feedback  
4. Click **Save** when done

### Step 2: Define Answer Input Fields

Use the next panel to input valid answers for each blank.

![Answer Fields](/screenshot/questions/fillintheBlankQ3.webp)

5. Write expected answers into the blank fields

### Step 3: Configure Timeline Behavior

Once the element is placed, it will appear on the layer:

![Layer View](/screenshot/questions/fillintheBlankQ4.webp)

1. Fill in the Blank Question element in the timeline  
Click the element to open detailed settings on the right panel.

### Step 4: Configure Right Panel Settings

Adjust how the question behaves on screen.

![Right Panel Settings](/screenshot/questions/fillintheBlankQ5.webp)

1. Element name  
2. When the question appears  
3. Skippable toggle  
4. Whether video should pause  
5. Visibility  
6. Remember answer state  
7. Confirmation button label

### Step 5: Advanced Display Settings

![Feedback Display Options](/screenshot/questions/fillintheBlankQ6.webp)

A. Customize feedback screen button text  
B. Persist user answer across project jumps  
C. Allow fast response (bypasses validation)

### Step 6: Define Response Actions

![Correct/Incorrect Response Handling](/screenshot/questions/fillintheBlankQ7.webp)

A. Action on correct answer  
B. Add another action  
C. Action on incorrect answer  
D. Add another incorrect action

### Step 7: Edit and Manage Content

![Edit Question Fields](/screenshot/questions/fillintheBlankQ8.webp)

8. Edit the written question  
9. Update the blank answers  
10. Add/edit feedback  
11. Save changes

</Steps>

<Callout type="important">
This element is case-sensitive. For better accuracy, avoid special characters in answers unless intentional.
</Callout>
