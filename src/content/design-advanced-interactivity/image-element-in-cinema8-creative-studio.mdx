# Image Element in Cinema8 Creative Studio

<Callout type="info">
The **Image Element** allows you to display static images during your video. You can also turn these into interactive buttons by assigning click actions.
</Callout>

## How to Add an Image Element

![Drag and drop Image element](/screenshot/advanced-interactivity/image-element-1.webp)

<Steps>

### Step 1: Add to Layer

Drag and drop the **Image** element from the Elements list into your video layer.

### Step 2: Browse the Library

Access Cinema8â€™s image library to choose from a collection of built-in visuals.

![Image gallery](/screenshot/advanced-interactivity/image-element-2.webp)

### Step 3: Search or Upload

Use the search bar to find specific images, or click **Upload** to add your own.

### Step 4: Insert the Image

Select your image and click **Add** to insert it into the timeline.

</Steps>

## Editing Image Element Settings

Once the image is placed, clicking it reveals the configuration panel.

![Image layer appearance](/screenshot/advanced-interactivity/image-element-3.webp)

### General Settings

| Option       | Description |
|--------------|-------------|
| **Name**     | Assign a label to identify the element. |
| **Start / End** | Define when the image appears and disappears. |
| **Visible**  | Choose if the image is shown by default. |

![Editing settings](/screenshot/advanced-interactivity/image-element-4.webp)

### Interactivity & Effects

| Option       | Description |
|--------------|-------------|
| **On Appear**     | Trigger one or more actions when the image enters the screen. |
| **On Exit**       | Define actions for when the image leaves the screen. |
| **On Click**      | Assign interactive behaviors when the image is clicked. |

![Click action options](/screenshot/advanced-interactivity/image-element-5.webp)

### Styling and Behavior

| Option       | Description |
|--------------|-------------|
| **Position & Size** | Set `Top`, `Left`, `Width`, and `Height` values (in percentages). |
| **Replace Image**   | Swap the image with another from the library. |
| **Click Flash Effect** | Enable a flashing visual when the user clicks the image. |
| **Screen Effects** | Add entry or exit animations to the image. |
| **Save / Delete**  | Save your changes or remove the element from the layer. |
