# How to Use Question Templates

## Adding and Customizing Question Templates in Your Videos

You can add questions to your videos in **Cinema8**, and customize the design of the question templates to better fit your content. Let's go through how to change and apply question templates.

<Steps>

### Step 1 — Open Customisation
Return to the main menu and click the **Customisation** button.

![Cinema8 Tools Button](/screenshot/questions/questionTemplate1.webp)

### Step 2 — Open Question Templates
Click the **Question Template** button on the next screen.

![Question Template Screen](/screenshot/questions/questionTemplate2.webp)

### Step 3 — Explore or Create Templates
On the screen, you'll see several options:

- **Search templates**
- **Your templates**
- **Default templates from Cinema8**
- **New Question Template** button
- **Settings** for each template

Click **New Question Template** to begin.

![Create New Question Template](/screenshot/questions/questionTemplate3.webp)

### Step 4 — Fill Out Template Info
- Enter a name
- Set the status (active/passive)
- Click to create the template

### Step 5 — Edit Template Design
From **My Templates**, click **Actions** > **Edit**. You’ll be taken to a screen where you can customize:

- HTML code
- CSS styles
- JavaScript interactions

![Edit Question Template](/screenshot/questions/questionTemplate4.webp)

### Step 6 — Apply Template to Videos
There are two ways to apply your template:

1. From the **Player Themes** in the Cinema8 Admin Panel — applies globally.
2. From the **Properties Panel** of any question element in the Studio — applies locally to one question.

![Cinema8 Player Themes](/screenshot/questions/questionTemplate5.webp)

</Steps>

<Callout type="info">
Selecting a question template from **Player Settings** applies it across all videos, while setting it from **Properties** affects only that individual question.
</Callout>
