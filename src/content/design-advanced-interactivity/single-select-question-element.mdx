# Single Select Question Element

<Callout type="info">
Use single-select questions to present viewers with one correct answer from a set of options — ideal for quizzes, polls, and assessments.
</Callout>

With the **Single Select Question** element, users can add questions that allow only one correct answer in their interactive videos. This is commonly used for quizzes or simple choice-based surveys.

## How to Add a Single Select Question

<Steps>

### Step 1: Drag the Element onto the Timeline

Open your video project in **Cinema8 Creative Studio** and drag the **Single Select Question** element from the Elements list into the timeline layer.

### Step 2: Configure the Question

Once added, a setup panel will appear. Complete the following fields:

1. **Question Name** – Used for internal reference and tracking
2. **Question Text** – What you want to ask the viewer
3. **Optional Images** – Add visuals to support your question
4. **Answer Options** – Enter possible answers and mark one as the correct answer

### Step 3: Save Your Settings

After configuring all details, click **Save** to add the question element to your interactive video.

</Steps>

<Callout type="default">
You can further enhance the question with feedback, logic-based navigation, and variables by opening advanced settings after placement.
</Callout>
