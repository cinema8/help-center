# Drag and Drop Question

<Callout type="info">
This question type allows users to match or sort visual items by dragging elements into designated areas. Ideal for learning, training, or gamified assessments.
</Callout>

## Adding Interactive Drag and Drop Questions to Your Cinema8 Videos

<Steps>

### Step 1: Add the Element

Drag and drop the **Drag and Drop Question** element from the **Elements** panel to the timeline.

![Initial Drop](/screenshot/questions/drag-and-drop-1.webp)

### Step 2: Configure Drag and Drop Details

![Question Settings](/screenshot/questions/drag-and-drop-2.webp)

1. **Name:** Internal reference for your question
2. **Add Drop Area:** Define areas where draggable items will be placed
3. **Drop Areas:** Review and adjust each drop zone
4. **Correct Answer:** Assign correct responses to each drop area
5. **Additional Options:** Add extra items (including distractors)
6. **Feedback:** Write response feedback
7. **Save:** Finalize your question

### Step 3: Open Element Settings Panel

Click the question element on the timeline to reveal its properties.

![Settings Panel](/screenshot/questions/drag-and-drop-3.webp)

1. **Element Name**
2. **Entry/Exit Time**
3. **Template Selection**
4. **Skippable Toggle**
5. **Pause Video on Display**
6. **Visibility Toggle**
7. **Remember Viewer’s Answer**

### Step 4: Define Action Behaviors

Use these controls to determine what happens after a viewer interacts:

![Action Settings](/screenshot/questions/drag-and-drop-4.png)

1. **Confirm Button Text** – What appears on the submit button
2. **Feedback Button Text** – Label for the post-answer screen
3. **Immediate Feedback** – Show a result without validation
4. **Correct Answer Action** – Define what happens on success
5. **Add Action** – Attach more responses
6. **Incorrect Answer Action** – Define outcome for incorrect answers

</Steps>

<Callout type="default">
For more about configuring actions, check the [Action Elements Guide](/design-advanced-interactivity/how-to-use-action-elements).
</Callout>
