# Free Text Question Element

## Create Open-Ended Questions for Audience Input

With the **Free Text Question** tool, users can create questions that allow the audience to write down their own answers.

<Steps>

### Step 1: Add the Element to the Timeline

Drag and drop the **Free Text Question** element from the **Elements** list onto your layer. The following screen will appear:

![Free Text Q1](/screenshot/questions/freeTextQ1.webp)

### Step 2: Configure the Question

![Free Text Q2](/screenshot/questions/freeTextQ2.webp)

1. Area to write the question.
2. Field where the answer will be entered.
3. Area to add feedback.
4. Click **Save** after completing your configuration.

### Step 3: Adjust the Element Settings

Once added, the element appears in your layer as shown below:

![Free Text Q3](/screenshot/questions/freeTextQ3.webp)

Click on the element to access its settings panel on the right side.

![Free Text Q4](/screenshot/questions/freeTextQ4.webp)

1. Name field of the added element.
2. Start and end time settings.
3. Skippable toggle.
4. Pause video when element appears.
5. Visibility toggle.
6. Option to remember previous answers.
7. Confirmation button text.

![Free Text Feedback Settings](/screenshot/questions/freeTextQ5.webp)

A. Feedback screen button text.
B. Option to preserve user answers between projects.
C. Quick answer mode (feedback disabled).

### Step 4: Define Actions Based on Answer

![Free Text Action Settings](/screenshot/questions/freeTextQ6.webp)

A. Action for correct answer.
B. Add new action.
C. Action for incorrect answer.
D. Add another action.

### Step 5: Edit and Save Final Settings

![Free Text Q7](/screenshot/questions/freeTextQ7.webp)

8. Edit the question content.
9. Answer input field.
10. Question option settings.
11. Save changes.
12. Delete the element.

### Step 6: Customize Feedback and Navigation

![Free Text Q8](/screenshot/questions/freeTextQ8.webp)

1. Positive feedback text.
2. Time to jump to when answered correctly.
3. Negative feedback text.
4. Time to jump to when answered incorrectly.

</Steps>

<Callout type="info">
Use Free Text Questions to gather user insights, open-ended feedback, or personal reflections as part of your interactive experience.
</Callout>
