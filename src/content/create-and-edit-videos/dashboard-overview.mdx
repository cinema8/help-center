# Dashboard Overview  
<span className="block text-gray-500 text-base font-normal mt-1">A visual guide to the Cinema8 dashboard</span>

Cinema8’s dashboard is your central hub for managing interactive video projects. It provides quick access to essential tools and features to streamline your video creation process.

---

## <span className="inline-flex items-center gap-2 text-gray-900 dark:text-white"><MapIcon className="w-6 h-6 text-blue-600" /> Key Sections</span>

- **Home:** Access tutorials, templates, and create new content quickly with options like **"New Video"**, **"Screen Recording"**, and **"360° Video"**.
- **My Videos:** Organize, edit, and manage your existing projects.
- **Analytics:** Track video performance, view heatmaps, and run A/B tests.
- **Help & Resources:** Click the ❓ icon for guides or to contact support.

## <span className="inline-flex items-center gap-2 text-gray-900 dark:text-white"><PlayCircleIcon className="w-6 h-6  text-blue-600" /> Video Creation & Management</span>

- Click **"New Video"** to start a project from scratch or template.
- Start **screen recordings** directly from the dashboard.
- Create immersive content with the **360° Video** tool.

---

## <span className="inline-flex items-center gap-2 text-gray-900 dark:text-white"><Cog6ToothIcon className="w-6 h-6  text-blue-600" /> Quick Tools & Shortcuts</span>

- Use **collaborative features** like commenting and sharing securely.
